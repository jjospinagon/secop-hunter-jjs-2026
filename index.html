Criterios de busqueda

Consulta directa a la API oficial de Colombia Compra Eficiente (datos.gov.co). No requiere CAPTCHA ni login. Para procesos abiertos hoy: estado Convocado + fecha cierre desde hoy.
Como se mapean los filtros
Dataset: p6dx-8zbt (SECOP II Procesos de Contratacion).
- Palabra clave -> $q= en todos los campos
- Entidad -> entidad like (texto) o nit_entidad si pones un NIT
- Depto/ciudad -> match exacto
- Modalidad/estado/fase/tipo -> match exacto
- Valor -> rango sobre precio_base
- Fechas -> rangos sobre publicacion y recepcion de ofertas
- URL -> campo urlproceso (enlace directo a SECOP II)
